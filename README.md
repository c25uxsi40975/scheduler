# 外勤調整システム

医員の外勤スケジュールを最適化するWebアプリケーションです。

## 機能

- **マスタ管理**: 医員・外勤先の登録・編集（役職、労力コスト、勤務時間等）、指名・限定メンバー・優先度設定、日別設定
- **希望入力**: 各医員が日程希望（○/当直明け○/△/×）・希望外勤先・日別外勤先希望・自由記述を入力
- **スケジュール自動生成**: PuLPによる3つの最適化モード
  - 案A: 給与均等重視（報酬のばらつきを最小化）
  - 案B: 希望重視（医員の希望を優先）
  - 案C: 優先度重視（◎○×設定を優先）
- **ML再調整**: RandomForestモデルで労力コストを考慮した最適な再割り当て
- **手動調整**: 生成された案の個別スロットを手動で変更可能（ハード制約違反は選択不可、ソフト制約違反は警告表示）
- **確定・共有**: 案を比較して確定、CSV出力
- **メール通知**: リマインダー・確定通知（Google Apps Script 連携）

## セットアップ

詳細なデプロイ手順は [docs/SETUP_GUIDE.md](docs/SETUP_GUIDE.md) を参照してください。

### ローカル開発

```bash
pip install -r requirements.txt
streamlit run app.py
```

ブラウザで `http://localhost:8501` が自動的に開きます。

**前提条件:**
- `.streamlit/secrets.toml` に GCP サービスアカウント情報とスプレッドシートキーを設定済みであること

## 使い方

### 管理者

1. 「管理者としてログイン」→ パスワード設定/入力
2. 「マスタ管理」で医員・外勤先を登録
3. 外勤先ごとに限定メンバー・優先度（必須/指名/任意/除外）を設定
4. 対象月を設定し、医員に希望入力を依頼
5. 「スケジュール生成」で3案を自動生成
6. 必要に応じて手動調整、または「ML再調整」でさらに最適化
7. 案を比較して確定

### 医員

1. 「医員としてログイン」→ アカウント名 + パスワード入力
2. 「希望入力」で日程希望（○/当直明け○/△/×）と希望外勤先を入力
3. 「スケジュール確認」で確定後のスケジュールを閲覧

## 制約条件

自動生成時・手動調整時の両方で以下の制約を考慮します。

### ハード制約（手動調整時は選択不可）

- 各外勤先に毎週/隔週で必要人数を割り当て（通常1人、2人体制対応）
- 休診に設定された日はスロットから除外
- 同一日に1人が複数の外勤先に行かない
- ×日（NG）には割り当てない
- 当直明け○の日はPM（午後）の外勤先のみ割当可
- 除外設定の外勤先には割り当てない
- 限定メンバーが設定された外勤先にはリスト内の医員のみ割当可
- 祝日は自動除外

### ソフト制約（手動調整時は警告表示、確認後に変更可能）

- △日にはペナルティ付きで割り当て可
- 日別外勤先希望との不一致を検出
- 月回数上限の超過を検出
- 必須設定の医員×外勤先は月1回以上割り当て
- 報酬のばらつき・外勤回数のばらつきを最小化
- 過去の全確定スケジュールの累計報酬を考慮
